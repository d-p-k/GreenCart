# 🛒 Green Cart

Green Cart is a modern e-commerce web application designed to connect users with fresh, eco-friendly and locally sourced products. The platform focuses on promoting sustainable shopping while providing a seamless and intuitive user experience.

## 🚀 Overview

Green Cart enables users to browse, search, and purchase products through a clean and responsive interface. The goal of the project is to build a scalable and user-friendly shopping platform with a focus on sustainability and usability.

## 🔥 Key Features

- 🛍️ Product browsing with clean UI
- 🔍 Search and filtering functionality
- 🛒 Add to cart & cart management
- 💳 Checkout flow (basic implementation)
- 👤 User authentication (if implemented)
- 📱 Fully responsive design
- ⚡ Fast and optimized performance

## 🏗️ Tech Stack

- Frontend: React, Tailwind CSS
- Backend: Java, Spring Boot
- Database: MySQL
- Tools: Razorpay, Cloudinary, Recharts, JWT
